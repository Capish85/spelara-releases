# Spelara

**A Windows desktop app for running and managing game servers.**

Spelara lets you install, start, stop, update, and monitor dedicated game servers from a single interface — no command line required.

---

## Download

Grab the latest installer from the [Releases](../../releases/latest) page.

---

## Requirements

- Windows 10 or Windows 11 (64-bit)
- An internet connection for installing servers via SteamCMD

No other dependencies — Node.js is bundled with the installer.

---

## Installation

1. Download `Spelara-Setup-x.x.x.exe` from the latest release
2. Run the installer and follow the wizard
3. Launch Spelara from the desktop shortcut or Start Menu

---

## What it does

- Install game servers via SteamCMD with a guided setup
- Start, stop, restart, and update servers from a simple dashboard
- Edit server configs directly in the app
- View live server logs
- Auto-restart servers on crash
- Supports servers that launch via batch files or custom scripts

---

## Notes

- Some servers require administrator privileges. Spelara will prompt you if needed.
- Server files are stored wherever you choose during setup. Uninstalling Spelara does not delete your servers.
